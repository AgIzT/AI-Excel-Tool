#### 🏁 **v1.0.0 AI-Excel-Tool 更新日志 "Reforged"** - 架构重构

架构：全面迁移至 PySide6 (Qt) 架构，引入多线程异步处理与现代化模块化 UI，修复原UI界面卡顿。

AI助手：新增专门的 AI 对话页面，支持现代化的聊天气泡 UI，可直接与 AI 助手进行自然语言交互解决单据异常。

界面：现代化 Clash 风格布局，侧边导航栏 + 主工作区路由架构，弃用旧版长滚动条设计。

日志：抽屉式可折叠日志面板，核心工作区视野更开阔，专注"单据流"本身。

视觉：统一全局 QSS 样式，扁平化无边框圆角高对比度暗色调设计。

***

#### 🏁 **v0.3 AI-Excel-Tool 更新日志** - 智能命名与模板管理

智能命名：AI 自动提取单据日期与供应商，生成「日期\_供应商\_入库.xlsx」智能文件名，支持冲突自动编号。

模板：新增模板管理器，支持多模板切换与自定义模板导入，内置默认模板与用户模板持久化。

架构：重构 DeepSeek 调用同时返回 meta 与 records，单次 API 调用完成信息提取，不增加成本。

兼容：完全向后兼容，AI 返回旧格式仍可正常解析，模板参数默认回退机制确保稳定性。

***

#### 🏁 **v0.2 AI-Excel-Tool 更新日志** - 功能完善

交互：支持多文件（图片/Excel）拖拽上传，新增列表预览及单文件删除功能。

识别：新增"手写体"开关。开启时优先提取手写涂改/备注；关闭则仅提取印刷体。

性能：支持 Excel 源文件直传，系统自动跳过 OCR 解析。

批处理：多任务异步处理不卡顿，新增"多单合并输出至单表"开关与结果核对弹窗。

#### 🏁 **v0.1 AI-Excel-Tool Beta (初代版本)** - 核心功能架构

初代版本跑通了业务全流程闭环，奠定了极简操作与 AI 自动化处理的基础。

【基础架构】 成功跑通GLM-OCR 识图提取 → DeepSeek 逻辑洗选与匹配 → 自动导出标准 Excel 表格全链路。

【极简操作流】 确立了左侧 01（选文件）- 02（AI 识别）- 03（打开导出文件）的"三步走"傻瓜式工作流。

【现代化 UI】 采用深色极简主题，提供清晰的图片预览面板。

【透明化反馈】 引入"实时处理日志"终端面板，将 AI 的请求、解析、报错等后台状态实时前台化，便于排查问题。

【内置模板】底层直接封装进货单导入模板，确保输出数据格式高度统一，可直接对接后续进销存系统。
